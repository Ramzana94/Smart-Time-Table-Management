# Smart Timetable Management System

## Overview

The Smart Timetable Management System is a Flutter and Firebase-based mobile application developed to automate academic timetable scheduling and resource management. The system helps educational institutions efficiently manage departments, teachers, courses, classrooms, and timetable generation while minimizing scheduling conflicts and improving administrative productivity.

## Key Features

* Department Management
* Teacher Management
* Course Management
* Classroom Management
* Automated Timetable Generation
* Conflict-Free Scheduling
* Firebase Authentication
* Cloud Firestore Integration
* Real-Time Data Management
* Responsive User Interface
* GetX State Management

## Technology Stack

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* GetX
* Material Design
  
## User Roles

### Administrator
- Manage Departments
- Manage Teachers
- Manage Courses
- Manage Classrooms
- Generate and Update Timetables
- Maintain Academic Records

### Faculty Members
- View Assigned Timetables
- Access Course Information
- Monitor Schedule Updates

### Students
- View Timetables
- Check Course Schedules
- Access Updated Academic Timings
